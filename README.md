### Chaotic Cold Diffusion Framework

A Cold Diffusion-based framework for modeling and generating chaotic sequences. This framework generalizes traditional diffusion models to work with chaotic systems like Logistic Map, Henon Map, and Lorenz System.

**[📖 Read the full documentation](CHAOTIC_DIFFUSION.md)**

#### Features
- Multiple chaotic system implementations
- Flexible degradation operators (noise, permutation, masking)
- Cold Diffusion model for chaotic sequences
- Visualization tools
- Example scripts

#### Quick Start

```bash
pip install -r requirements.txt
python chaotic_diffusion/examples/basic_example.py
```
